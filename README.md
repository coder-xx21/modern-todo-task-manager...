# TaskFlow Manager

A modern, responsive To-Do Manager website built with HTML, CSS, and JavaScript.

## Features
- Modern landing page with navigation
- Dashboard for managing tasks
- Add, edit, delete, and mark tasks as complete or pending
- Search and filter tasks by priority and status
- Sort tasks by due date or priority
- Task statistics for total, completed, and pending items
- Local storage support so tasks remain after refresh
- Light/Dark mode toggle
- Smooth animations and responsive design

## Project Structure
- index.html - Main structure of the website
- style.css - Styling and responsive layout
- script.js - Task logic, filtering, local storage, and theme toggle

## How to Run
1. Open the project folder in your browser.
2. Open index.html directly, or use a simple local server.

Example with VS Code Live Server:
- Install the Live Server extension
- Right-click index.html
- Select "Open with Live Server"
